# Student Management System

A simple Student Management System built using **Django** and **HTML**.

## Features
- Add Student
- View Student List
- Update Student
- Delete Student
- Search Students
- Admin Panel

## Technologies Used
- Python
- Django
- HTML
- SQLite

## How to Run the Project
1. Clone the repository
2. Create a virtual environment
3. Install Django
4. Run migrations
5. Start the server
6. http://127.0.0.1:8000/

```bash
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
